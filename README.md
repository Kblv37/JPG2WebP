# JPG → WEBP Converter

Лёгкий браузерный конвертер JPG в WEBP.

##  Возможности

- Конвертация изображений на стороне клиента
- Выбор качества (100–30)
- Не требует backend
- Мгновенное скачивание результата
- Файлы не отправляются на сервер

##  Как это работает

Изображение конвертируется прямо в браузере с помощью Canvas API.  
Вся обработка происходит локально — данные не покидают ваше устройство.
